# Financial AI Analyst 📊

כלי AI לניתוח דוחות כספיים רבעוניים בעברית — מאת **דורון שרייבמן**

## מה הכלי עושה

- מעלה דוח רבעוני PDF ומנתח אותו אוטומטית
- מציג KPIs מרכזיים: הכנסות, רווח נקי, EPS, תזרים, מרווחים
- מייצר ניתוח נרטיבי מקצועי בעברית בסגנון אנליסט וול סטריט
- מזהה beat/miss מול תחזיות קונסנזוס
- כולל סוכן אימות שמצליב נתונים מול המסמך המקורי
- ממשק שיחה לשאלות על הדוח

## איך משתמשים

1. פתח את `index.html` בדפדפן (או דרך GitHub Pages)
2. הזן **Gemini API Key** מ-[Google AI Studio](https://aistudio.google.com/apikey) — חינמי
3. העלה דוח PDF
4. לחץ "נתח דוח עכשיו"

## דרישות

- דפדפן מודרני (Chrome / Edge מומלץ)
- Gemini API Key (חינמי)
- אין צורך בהתקנה
